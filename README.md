# Paynotes
An application to save edit and delete notes on the server.

# Tech Stack:

Front end : ReactJS/ Redux
Back end: NodeJS/ ExpressJS
Database: MongoDB hosted on mongolabs

# To get started:

Run these commands :

git clone <repo>

cd paynotes/

npm install

# For development :- 

npm run dev-server

npm run build

# For production :-

npm start
